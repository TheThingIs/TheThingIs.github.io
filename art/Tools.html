Tools I use

Medibang

Medibang is the main drawing software that I use to create my art. The main reason why I love the program is becase of the correction/stabelicer tool because it helps a lot when it comes to drawing. Medibang is a free software that is overall easy to use

Wacom Cintiq

Currently, I make all my art on a wacom cintiq 13hd. It a drawing tablet with a screen wich makes it possible to see what the pen is doing where it is. Bofore I had my cintiq I drew on a "one by wacom". The tablet did the work quite well but I am a lot better with the cintiq. My oldest artwork was drawn with my finger on an ipad.
